### Deepin 15 右键刷新菜单插件

##### 一键安装

1. 硬件使用F5刷新

`sh -c "$(wget https://raw.githubusercontent.com/Lyvnee/deepin-refresh/master/install1.sh -O -)"`

2. 硬件使用Fn+F5刷新

`sh -c "$(wget https://raw.githubusercontent.com/Lyvnee/deepin-refresh/master/install2.sh -O -)"`

##### 手动安装

下载文件执行`install.sh`
